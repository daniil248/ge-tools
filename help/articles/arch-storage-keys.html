Соглашение об именовании

Все per-project ключи имеют префикс raschet.project.<pid>. где <pid> — id проекта.

Главные ключи

КлючСодержимое
raschet.project.<pid>.engine.scheme.v1JSON схемы (Конструктор)
raschet.project.<pid>.por.objects.v1POR-объекты
raschet.project.<pid>.tech-workspace.variants.v1Варианты концепции
raschet.project.<pid>.tech-workspace.activeVariantId.v1id активного варианта
raschet.project.<pid>.rack-config.instances.v1Стойки (legacy)
raschet.project.<pid>.rack-config.tombstones.v1Tombstones (удалённые id)
raschet.project.<pid>.scs-config.contents.v1Содержимое стоек
raschet.project.<pid>.scs-config.rackTags.v1Тэги стоек СКС
raschet.project.<pid>.scs-design.links.v1Меж-шкафные связи
raschet.project.<pid>.scs-design.plan.v1План зала

Глобальные (не per-project)

raschet.projects.v1Список всех проектов (Local-режим)
raschet.storageMode.v1Режим хранения user override (auto/local/cloud)
raschet.cardPresets.v1User-пресеты карточек
raschet.cardPresetActive.v1id активного user-пресета
raschet.elementLibrary.v1User-каталог изделий

Helper-функция

// shared/project-storage.js
import { projectKey } from './project-storage.js';
projectKey('p_123', 'engine', 'scheme.v1')
// → 'raschet.project.p_123.engine.scheme.v1'

Все модули используют этот helper — единый источник истины для именования.
