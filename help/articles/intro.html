Назначение

GE Tools (Genesis Engineering Tools) — набор инженерных и управленческих модулей и инструментов для широкого спектра проектных задач: электроснабжение, СКС, холодоснабжение, ГОТВ, метео, логистика, сервис/ТО, технолог-ГИП, отчёты и др. От концепции (предпроектная стадия) до детальных схем, спецификаций и эксплуатации. Подходит для ЦОД, промплощадок, складов, инженерных систем зданий. Конструктор схем — один из модулей.

Ключевые возможности

  • Принципиальная схема с автоматическим подбором кабелей и автоматов (ПУЭ / IEC).
  • Расчёт нагрузок, токов короткого замыкания, падения напряжения, селективности.
  • Концепция объекта (предпроект) — мощности, площади, multi-variant compare.
  • Конфигураторы: ИБП, НКУ, РУ СН, трансформатор, СКС-стойка, PDU, АГПТ.
  • Каталог изделий с импортом прайс-листов и аналитикой.

Параллельная работа команды

В GE Tools нет жёсткого «конвейера». Все специалисты работают над одним проектом одновременно и видят изменения друг друга. Возможны два сценария:

  1. Технолог задаёт концепцию (количество стоек, мощность, профиль) → дальше электрик, СКС-инженер, климатолог, логист работают параллельно над общим объектом, каждый в своём домене.
  2. Все работают одновременно с самого начала, без явного «handoff». Концепцию уточняем по ходу — GE Tools поддерживает живую синхронизацию через Firebase + cross-discipline reconciliation (один физический объект, разные параметры по доменам).

Все участники видят актуальное состояние проекта. Конфликты атрибутов решаются в инспекторе («🔀 Похожие группы», cross-discipline reconciliation).

С чего начать

  1. Откройте «Проекты» — создайте проект (или используйте default).
  2. Если у вас концептуальная стадия — заходите в Технолог ЦОД.
  3. Если уже есть ТЗ и нужна схема — открывайте Конструктор схем.
  4. Каждый модуль имеет встроенный значок ? с подсказкой по работе.

Режим хранения

Все данные могут храниться локально (LocalStorage) или в облаке (Firebase). По умолчанию — авто: облако если есть Gmail-вход, иначе локально. В шапке инструментов есть переключатель режима.

💡 Совет: для одиночной работы выбирайте локальный режим — нет ограничений на запись (Firebase бесплатный тариф ограничен квотами).
