Назначение

Кнопка 📄 Пояснительная записка в левой панели tech-workspace формирует структурированный HTML-документ по активному варианту концепции. Документ можно сохранить в PDF через native Ctrl+P → «Сохранить как PDF» (родная функция всех современных браузеров).

Структура документа

  1. Описание объекта — summary с ключевыми параметрами
  2. Концепция стоек — таблица групп с профилем / count / kW / PDU / размеры
  3. Электроснабжение (ИБП) — таблица систем + Σ доступной мощности (split IT / climate)
  4. Климат — таблица групп + предупреждение если холода меньше IT
  5. Ввод (ТП и ДГУ) — параметры + Σ принятая мощность
  6. Площади помещений — таблица по ТКП 308-2011 / TIA-942
  7. Перечень ТЗ для смежных дисциплин:
    • Электрик: ИБП, автоматы, кабели, ГРЩ, ТП, ДГУ
    • СКС-инженер: расположение стоек, кабельные трассы
    • Климатик: подбор кондиционеров, проверка запаса
    • Архитектор: компоновка помещений, пожарная безопасность

Стили

Документ оптимизирован под печать:

  • @page { size: A4; margin: 20mm; }
  • font-family: "Times New Roman" — стандарт для технической документации
  • Таблицы с border, выравниванием чисел справа
  • Кнопки 🖨/✕ скрываются при печати (@media print)
