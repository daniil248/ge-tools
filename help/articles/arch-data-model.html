Слои хранения

  1. engine.scheme.v1 — JSON-схема (узлы, связи, страницы, режимы). Главный artifact Конструктора.
  2. POR objects (por.objects.v1) — объектная модель проекта (rack, panel, consumer-system и т.п.). Domains: mechanical / electrical / scs / location / hvac.
  3. tech-workspace.variants.v1 — концепции технолога. Массив вариантов с rackGroups[], upsSystems[], coolingUnits[], feed{tp,dgu}.
  4. scs-config.* / scs-design.* — данные СКС-инженера.
  5. rack-config.instances.v1 — legacy storage стоек.

Связи между слоями

  • engine consumer-rack ↔ scs-config rack: bridge через shared/scheme-rack-bridge.js (виртуальные id scheme-<nodeId>-<i>).
  • POR rack ↔ engine consumer: n.porObjectId + obj.legacyRackId.
  • Tombstones (anti-resurrection): см. scs-config.
