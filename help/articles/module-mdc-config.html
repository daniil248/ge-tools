Назначение

Модульный ЦОД (МЦОД) — конфигуратор для специализированных серий типа GDM-600. Wizard зон, расстановка стоек / ИБП / кондёров.

Когда использовать

  • Объект — модульный ЦОД (контейнерное / blade-исполнение)
  • Уже есть готовая серия (GDM-600 и т.п.) с известными модулями

Связи

  • Технолог ЦОД — на стадии концепции, MCD-вариант
  • В будущем — handoff из tech-workspace в mdc-config с предзаполнением
