Какой модуль когда использовать

Платформа разделена по этапам и ролям. Каждый модуль — отдельный SPA-вход с шапкой, где есть кнопка ← возврата к проекту/предыдущему модулю.

Связи между модулями

Модули обмениваются данными через общие хранилища (см. архитектура):

  • Технолог → Конструктор: handoff (см. статью) — концепция превращается в узлы схемы.
  • Конструктор → Компоновщик СКС: consumer-узлы со subtype=rack автоматически попадают в список стоек СКС (через scheme-rack-bridge).
  • Конструктор → Расчёт кабеля: щелчок по линии открывает калькулятор с предзаполненными I, U, длиной.
  • Каталог → все: picker'ы изделий используют общую библиотеку shared/element-library.js.
